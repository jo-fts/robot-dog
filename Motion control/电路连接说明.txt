# 四足机器人电路连接功能与使用说明

## 一、硬件组件清单

| 组件名称 | 数量 | 用途 |
|---------|------|------|
| STM32F103C8T6开发板 | 1 | 主控制器 |
| MG996R舵机 | 12 | 腿部关节驱动 |
| HC-05蓝牙模块 | 1 | 无线通信 |
| OLED显示屏(128x64) | 1 | 显示表情与状态 |
| 电源模块(7.4V锂电池) | 1 | 主电源 |
| 舵机驱动板 | 1 | 舵机电源管理 |
| 连接线 | 若干 | 电路连接 |

| PA9 | USART1_TX | 语音模块(可选) |
| PA10 | USART1_RX | 语音模块(可选) |
| PB10 | USART3_TX | HC-05蓝牙模块 |
| PB11 | USART3_RX | HC-05蓝牙模块 |
| PB6 | I2C1_SCL | OLED显示屏 |
| PB7 | I2C1_SDA | OLED显示屏 |
| VCC | 电源(3.3V) | 蓝牙模块、OLED |
| GND | 地线 | 所有组件 |

## 三、电路连接图

```
+---------------------+
| STM32F103C8T6       |
|                     |
|  PA0-7  PWM输出      |-----> 舵机1-8
|  PB0-3  PWM输出      |-----> 舵机9-12
|  PB10   USART3_TX    |-----> HC-05 RX
|  PB11   USART3_RX    |<----- HC-05 TX
|  PB6    I2C1_SCL     |-----> OLED SCL
|  PB7    I2C1_SDA     |-----> OLED SDA
|  VCC(3.3V)           |-----> 蓝牙、OLED VCC
|  GND                 |-----> 所有组件GND
+---------------------+

+---------------------+
| 电源模块(7.4V)      |
|                     |
| 正极                |-----> 舵机驱动板VCC
| 正极                |-----> STM32 VIN
| 负极                |-----> 所有组件GND
+---------------------+

+---------------------+
| 舵机驱动板          |
|                     |
| VCC(7.4V)           |<----- 电源模块
| GND                 |-----> 所有组件GND
| 舵机1-12接口        |<----- 对应舵机
+---------------------+
```

## 四、使用说明

### 1. 硬件连接步骤

1. **连接电源模块**
   - 将7.4V锂电池连接到电源模块
   - 电源模块正极连接到STM32的VIN引脚和舵机驱动板的VCC
   - 电源模块负极连接到所有组件的GND

2. **连接舵机**
   - 将12个舵机按照编号对应连接到STM32的PWM输出引脚
   - 舵机的信号线连接到STM32的PWM引脚
   - 舵机的电源线连接到舵机驱动板
   - 舵机的地线连接到GND

3. **连接蓝牙模块**
   - HC-05的VCC连接到STM32的3.3V
   - HC-05的GND连接到GND
   - HC-05的TX连接到STM32的PB11(USART3_RX)
   - HC-05的RX连接到STM32的PB10(USART3_TX)

4. **连接OLED显示屏**
   - OLED的VCC连接到STM32的3.3V
   - OLED的GND连接到GND
   - OLED的SCL连接到STM32的PB6
   - OLED的SDA连接到STM32的PB7

### 2. 软件配置

1. **编译下载代码**
   - 使用Keil MDK打开项目文件
   - 编译代码确保无错误
   - 通过ST-Link或串口下载代码到STM32

2. **蓝牙配对**
   - 打开手机蓝牙
   - 搜索并连接名为"HC-05"的设备
   - 配对密码默认为"1234"

3. **WeXCube配置**
   - 打开WeXCube小程序
   - 搜索并连接四足机器人设备
   - 在设备控制页面添加对应控件

### 3. 功能测试

1. **基本测试**
   - 上电后OLED应显示休眠表情
   - 蓝牙模块指示灯应闪烁
   - 舵机应回到初始位置

2. **动作测试**
   - 在WeXCube小程序中测试各个动作按钮
   - 检查机器人是否能正常执行前进、后退、转弯等动作
   - 测试身高调节功能

3. **蓝牙通信测试**
   - 检查手机与机器人的连接状态
   - 测试指令响应速度

### 4. 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|----------|
| 舵机不转 | 电源电压不足 | 检查电源连接，确保电压≥7.4V |
| 蓝牙无法连接 | 模块未配对 | 重置HC-05模块，重新配对 |
| OLED无显示 | I2C连接错误 | 检查PB6、PB7引脚连接 |
| 动作异常 | 舵机角度设置错误 | 检查Servo.c中的角度范围设置 |
| 程序无响应 | 硬件故障 | 检查STM32复位电路，重新下载程序 |

## 五、注意事项

1. **电源安全**
   - 请勿使用超过8.4V的电源
   - 长时间使用时注意电池温度
   - 不使用时请断开电源

2. **舵机保护**
   - 避免舵机长时间卡住
   - 动作执行时注意周围环境，防止碰撞

3. **蓝牙使用**
   - 蓝牙通信距离建议在10米以内
   - 避免在强电磁干扰环境下使用

4. **程序更新**
   - 更新程序前请备份原有代码
   - 确保STM32处于正常工作状态

## 六、故障排除流程

1. **硬件检查**
   - 检查所有连接是否牢固
   - 测量电源电压是否正常
   - 检查各模块是否损坏

2. **软件检查**
   - 检查代码编译是否有错误
   - 检查串口通信是否正常
   - 检查中断处理是否正确

3. **功能测试**
   - 逐步测试各个功能模块
   - 记录测试结果和错误信息
   - 根据错误信息定位问题

## 七、维护保养

1. **定期检查**
   - 检查舵机连接线是否松动
   - 检查电池电量和状态
   - 清理机器人内部灰尘

2. **舵机维护**
   - 定期给舵机转轴添加润滑油
   - 避免舵机进水或进尘

3. **电池维护**
   - 锂电池应定期充放电
   - 长期不使用时应保持50%电量
