# 机器狗动作速度调整计划

## 一、速度控制参数分析

通过代码分析，发现以下控制动作速度的关键参数：

### 1. 延迟时间参数 (tt)
- **前进/后退**：当前值为 8ms
- **左转/右转**：当前值为 6ms
- **摇摆**：当前值为 50ms
- **跳舞**：固定 1000ms 延迟

### 2. 动作步长参数 (speed_value)
- 所有动作函数中默认值为 0.02
- 值越大，动作越快

### 3. 单轴动作延迟
- `Action_dongzhuo_1` 和 `Action_dongzhuo_2` 中使用 `step_delay = t / 2`
- 跳舞动作中使用固定延迟

## 二、调整方案

### 1. 调整延迟时间参数 (tt)
- **前进/后退**：从 8ms 调整为 4ms
- **左转/右转**：从 6ms 调整为 3ms
- **摇摆**：从 50ms 调整为 25ms
- **跳舞**：保持 1000ms 不变（需要明显的动作间隔）

### 2. 调整动作步长参数 (speed_value)
- 将所有动作函数中的 `speed_value` 从 0.02 调整为 0.04
- 这会使动作周期缩短一半

### 3. 调整单轴动作速度
- 保持 `step_delay = t / 2` 的计算方式
- 但在调用这些函数时，可以适当减小 `t` 参数值

## 四、注意事项

1. **不要过度加快**：动作过快可能导致舵机响应不及时，影响稳定性
2. **逐步调整**：如果动作仍然不够快，可以进一步减小延迟时间
3. **保持平衡**：确保所有动作的速度协调一致
4. **监控温度**：动作加快会增加舵机负载，注意监控温度

- 转弯动作更加迅速
